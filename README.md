# MetaForge SEO — Shopify App

AI-powered SEO automation for Shopify stores. Auto-generates optimized meta titles, descriptions, JSON-LD structured data, and blog content from your product data.

## What It Does

- **Dashboard** — SEO health score, store stats, and bulk generation with real-time progress
- **Meta Generator** — Generates and applies SEO meta titles (≤60 chars) and descriptions (≤155 chars) for products (AI when configured, template fallback)
- **Schema Markup** — Builds and applies valid JSON-LD structured data (schema.org/Product) for Google rich snippets
- **Blog Generator** — AI-powered blog posts (buying guides, product spotlights, comparisons) via Claude API
- **Settings** — Encrypted Anthropic API key management and theme integration docs

## Prerequisites

- Node.js 20.19+ (or 22.12+) — per `package.json` engines
- A [Shopify Partner account](https://partners.shopify.com) (free)
- A development store (create one from your Partner dashboard → Stores)
- An Anthropic API key — only needed for Blog Generator; Meta/Schema features work without it

## Setup

### 1. Install dependencies

```bash
npm install
```

### 2. Set up environment variables

Create a `.env` file. The Shopify CLI will generate `SHOPIFY_API_KEY`, `SHOPIFY_API_SECRET`, and `SHOPIFY_APP_URL` automatically on first run, but you must add:

```env
ENCRYPTION_KEY=<generate below>
```

Generate your encryption key:

```bash
node -e "console.log(require('crypto').randomBytes(32).toString('hex'))"
```

### 3. Initialize the database

```bash
npx prisma db push
```

### 4. Start the dev server

**First time** (interactive — run in your terminal, not via script):

```bash
shopify app dev --reset
```

This prompts you to log in to your Partner account, select or create an app, and choose a dev store. It then starts a tunnel and prints an install URL.

**After initial setup:**

```bash
shopify app dev
```

Press `P` to open the app URL in your browser. Click **Install** in the Shopify admin.

## Environment Variables

| Variable | Required | Description |
|---|---|---|
| `SHOPIFY_API_KEY` | Yes (auto) | Generated by Shopify CLI |
| `SHOPIFY_API_SECRET` | Yes (auto) | Generated by Shopify CLI |
| `SCOPES` | Yes (auto) | `read_products,write_products,read_content,write_content` |
| `ENCRYPTION_KEY` | Yes | 64-char hex string — encrypts stored API keys |
| `DATABASE_URL` | Production only | PostgreSQL URL (SQLite used in dev) |
| `SHOP_CUSTOM_DOMAIN` | Optional | Allow custom shop domains (e.g. `example.com`) |

## Theme Integration (for JSON-LD Schema)

After generating and applying schema markup, add this to your theme's `sections/main-product.liquid` (inside the `<head>` or just before `</body>`):

```liquid
{% if product.metafields.metaforge_seo.json_ld %}
  <script type="application/ld+json">
    {{ product.metafields.metaforge_seo.json_ld.value }}
  </script>
{% endif %}
```

Validate your schema at [Google's Rich Results Test](https://search.google.com/test/rich-results).

## Testing

1. Open the app in your dev store — you'll see the SEO Health Dashboard
2. Go to **Meta Generator** → select products → **Generate Meta Tags** → review → **Apply to Shopify**
3. Verify in Shopify Admin → Products → [Product] → scroll to "Search engine listing"
4. Go to **Schema Markup** → generate and apply JSON-LD
5. Test JSON-LD at [Google's Rich Results Test](https://search.google.com/test/rich-results)
6. For **Blog Generator**: add your Anthropic API key in **Settings** first, then select up to 5 products and generate
7. From the **Dashboard**, use "Generate All Missing Meta" / "Generate All Schema" for bulk operations with progress tracking

## Deployment

### Switch to PostgreSQL

1. Update `prisma/schema.prisma` datasource:

```prisma
datasource db {
  provider = "postgresql"
  url      = env("DATABASE_URL")
}
```

2. Set `DATABASE_URL` in your hosting environment
3. Run `npx prisma migrate deploy`

### Hosting options

- [Google Cloud Run](https://shopify.dev/docs/apps/launch/deployment/deploy-to-google-cloud-run) — recommended, detailed tutorial
- [Fly.io](https://fly.io/docs/js/shopify/) — quick single-machine deploy
- [Render](https://render.com/docs/deploy-shopify-app) — Docker-based
- [Manual deployment guide](https://shopify.dev/docs/apps/launch/deployment/deploy-to-hosting-service)

Set `NODE_ENV=production` in your hosting environment.

### Key Prisma models

| Model | Purpose |
|---|---|
| `Session` | Shopify auth sessions (managed by library) |
| `SeoJob` | Bulk operation tracking (status, progress, errors) |

| `AppSettings` | Per-shop AI API key (encrypted) and model selection |
| `GeneratedContent` | Blog articles — title, meta, markdown body, publish status |

## Development Notes

- All app routes are prefixed `app.` (e.g. `app.meta-generator.tsx`) to inherit auth from `app/routes/app.tsx`
- Every loader/action calls `await authenticate.admin(request)` — handles OAuth redirect automatically
- Shopify Admin API version: `ApiVersion.October25` (2025-10)
- Webhook API version lives in `shopify.app.toml` (currently `2026-04`) — keep these aligned when upgrading
- Polaris web components are used (`<s-page>`, `<s-card>`, `<s-button>`, etc.) — not React Polaris
- The `shopify app dev` command **must be run interactively** in a real terminal — it prompts for Partner org and dev store selection
- Schema is applied as metafield `metaforge_seo.json_ld` (type: `json`) on each product
- Dashboard bulk operations are template-based (AI generation only happens in the Meta Generator when a key is configured)

## Troubleshooting

**Database tables don't exist:**
```bash
npx prisma db push
```

**App won't install / OAuth loop:**
Make sure `shopify app dev` is running and the tunnel URL matches your app's URLs in Partners dashboard.

**Blog generation fails:**
- Check your Anthropic API key in Settings → Test Connection
- Ensure `ENCRYPTION_KEY` is set in `.env` (must be 64 hex chars)

**Schema not showing in Google:**
- Confirm the Liquid snippet is in your theme (see Theme Integration above)
- Run the Rich Results Test — it takes a few days for Google to re-crawl
